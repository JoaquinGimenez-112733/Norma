# Norma

## Controles

- **A** – Mover a la izquierda  
- **D** – Mover a la derecha  
- **CTRL** – Agacharse  
- **Espacio** – Saltar  
- **ESC** – Salir del juego
- **R** – Restart    
- **E** – Interactuar ?? not yet
